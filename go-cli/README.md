# elering-dashboard-cli

AQL-driven CLI and REPL for the EleringDashboard SDK. Positional arguments are
joined into a single AQL expression and evaluated; with no arguments,
falls into an interactive REPL.

Built on `github.com/aql-lang/aql/eng/go` and the sibling Go SDK at `../go`.

## Build

```sh
go build -o elering-dashboard-cli ./...
```

## Run

```sh
# One-shot: arguments form a single AQL expression
./elering-dashboard-cli load 1 balance
./elering-dashboard-cli load '{id:1}' balance

# REPL
./elering-dashboard-cli
```

## Words

| Word     | Signatures                                   | Description                |
|----------|----------------------------------------------|----------------------------|
| `load`   | `[entity]` · `[query entity]`                | Load a single record       |

`query` is either a Map (`{id:1}`) or a Scalar (`1`, treated as `{id:1}`).
`entity` is one of the SDK's entity names (auto-quoted as an atom).

## Entities

balance balance_controller firm firm_capacity_controller gas_balance_controller gas_border_trade_controller gas_system gas_system_controller gas_trade gas_trade_controller gas_transmission_controller green_controller interruptible interruptible_capacity_controller nomination nominations_controller nps_controller renomination renominations_controller system system_controller transmission_controller umm_gas_controller umm_rss_feed_controller

## REPL commands

- `:quit` / `:q` / `:exit` — exit the REPL
- `:help` / `:h` / `:?`     — show help
